06 비구조화 할당 (구조분해) 문법 

객체 안의 값 추출 해서 바로 변수 혹은 상수로 선언하거나
함수의 파라미터에서 바로 객체의 값을 상수형태로 받아올수 있음.

만약 객체안에 존재하는 값이 없을땐?

1. 비구조화 할당시 기본값 설정

함수의 파라미터와 상수선언 모두 이용가능.

2. 비구조화 할당시 이름 바꾸기 

이름이 같다면 그냥 바로 비구조화 활당을 쓰면 되는데 이름이 서로 다름.
이럴 때 : 문자를 사용해 이름 변경.

const { name: nickname } = animal;

3. 배열 비구조화 할당 

배열안에 있는 원소를 다른 이름으로 새로 선언해주고 싶을때
사용하기 매우 유용. 기본값 지정도 가능 .

4. 깊은 값 비구조화 할당

객체의 깊은 곳에있는 값들을 밖으로 꺼내주고 싶다면 어떻게?
두가지 해결방법 존재.

4-1 sol 1 비구조화 할당 여러번 사용하기

key 선언시 선언된 값이 존재하고, value 를 입력하지않으면,
바로 매칭시켜주는 문법이 있음. ES6 - object-shorthand 문법 

deepObject 객체에서 각 key 에 해당하는 값을 추출하는 과정에서
비구조화를 여러번 했다.

4-2 sol 2 비구조화 할당으로 한번에 모두 추출하기.

깊은 객체의 형식을 따라가면서 추출.

편한방식으로 사용하자.
