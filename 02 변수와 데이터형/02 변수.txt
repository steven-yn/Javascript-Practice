변수 

let 은 변수, const 는 상수 

var 은 모오오오오오던 자바스크립트 에서는 사용 x

var 이 let 과 다른 주요 차이점?
똑같은 이름으로 여러번 선언 가능.
var 과 let 이 사용할수 있는 범위가 다르다.

IE9, IE10 같은 구형 브라우저에서는 let 과 const 를
지원하지 않는다.
하지만, 보통 개발할시엔 babel 같은 도구를 사용해서 
코드가 구형 브라우저에서도 돌아갈수 있게 변환작업 함.

데이터 타입

1. 숫자 
숫자는 그냥 바로 대입

2. 문자열 
텍스트(문자열) 형태의 값은 작은 따옴표 혹은 큰 따옴표로 감쌈.

3. 참/거짓

boolean 은 참 거짓 두가지 종류의 값만을 나타냄 

4. null과 undefined

자바스크립트 에선 '없음' 을 의미하는 데이터 타입이 두종류 있음.
null과 undefined 이 있는데, 둘의 용도가 살짝 다름.

null 은 주로 이 값이 없다! 라고 선언할때 사용 
undefined 는 아직 값이 설정되지 않은것을 의미.
