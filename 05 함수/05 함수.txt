05 함수 

함수는 특정 코드를 하나의 명령으로 실행할수 있게 해줌.

function 키워드 사용, 어떤 값 받아올지 정해주는것은
파라미터 매개변수 인자값 아규먼트 등 이라고 함.

함수 내부에서 return 키워드를 사용해 함수의 결과물을 지정.

return 하게되면 함수가 끝나버림.

1. 템플릿 리터럴

문자열 조합을 하기위해 + 연산자를 사용할수 있다.
더욱 편하게 조합하려면? es6의 템플릿 리터럴 문법 사용.

`string ${parameter} string`

function 키워드 대신 => 사용 .
