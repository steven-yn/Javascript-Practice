04 조건문 

1-1 if 문 

if 문 사용하면 특정 조건이 만족될 때만 특정 코드 실행.

if (조건) {
  코드;
}

조건이 만족 됐을 때 시행시킬 코드가 {} 로 감싸졌는데,
이것을 코드블록 이라고 한다.

조건이 true 가 되면 코드가 실행되고
false 가 된다면 실행되지 않음.

let 과 const 는 다른 블록 범위에서는 똑같은 이름으로 선언이 된다.

코드 블록 내부와 바깥의 let 과 const 는 이름이 같더라도
완전히 다른것
