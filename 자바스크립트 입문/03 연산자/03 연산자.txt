03 연산자 

= 는 대입 연산자에 해당한다.

새로운 변수를 선언 하는 = 는 대입 연산자가 아님.
변수에 값을 대입 하는 = 가 대입연산자.

	1. 산술 연산자 
	
	산술 연산자는 사칙연산.
	
	+ - * /
	
	변수에 ++ 하면 바로 1을 더해줌.
	그런데 ++ 가 변수 이름앞에 오는것과 뒤에오는것 차이점?
	
	console.log(c++); <- 더하기 직전값 출력
	console.log(++c); <- 더한 다음의값 출력
	
	뺄셈도 똑같이 할수있음.
	
	2. 대입 연산자 
	
	특정 값에 연산 한값을 바로 설정할때 사용
	
	= += -= *= /=
	
	3. 논리 연산자 
	
	불린 타입 을 위한 연산자. if 문 사용시 유용함.
	
	! && || ( not and or )
	
	- AND
	양쪽의 값이 둘다 true 일때만 결과물이 true 가 된다.
	
	- OR
	양쪽의 값이 둘다 false 일때만 결과물이 false 가 된다.
	
	* 연산 순서 
	
	논리 연산자의 연산 순서는 NOT - AND - OR 
	
	4. 비교 연산자
	
	비교 연산자는 두 값을 비교할때 사용.
	
	두 값이 일치하는가? === 사용 
	
	== 로도 사용할수는 있지만, === 과의 차이점은
	== 는 타입 검사까지는 하지 않는다.
	== 를 사용하면 숫자 1과 문자'1' 이 동일한 값으로 간주 된다.
					0과 false 도 동일한 값으로 간주 된다. 
					undefined 와 null 도 같은 값으로 간주 된다.
	
	상수 또는 변수등을 선언해서 비교연산자 돌려 나온 불린값을 대입 가능.
	
	=== 사용을 권장.
	
	두 값이 일치하지 않는가? !== 사용 
	
	!= 사용시 타입검사를 하지않음.
	
	크고 작음
	
	> < >= <= 
	
	문자열 붙이기
	
	문자열을 붙일때엔 + 로 할수 있다.
	string 을 객체에담아 더할수 있다. ㄷㄷ
